Creating a new git repo



1. cd /path/to/your/existing/code


2. echo "# heyyyy" >> README.md


3. git init


4. git add README.md


5. git commit -m "first commit"


6. git remote add origin https://github.com/Acespace/heyyyy.git


7. git push -u origin master

Congrats yo!
